<h2>Overview</h2>

<p>
This <strong>Weather App</strong> is built using <code>HTML</code>, <code>CSS</code>, and <code>JavaScript</code>.
It delivers real-time weather data powered by the <a href="https://openweathermap.org/api" target="_blank">OpenWeather API</a>.
The interface is designed with a clean <strong>light blue gradient</strong>, responsive layout, and smooth transitions for an enjoyable user experience.
</p>

---

<h2>Features</h2>

<ul>
  <li>Search for weather by city name</li>
  <li>Displays temperature, humidity, and weather conditions</li>
  <li>5-day forecast with weather icons</li>
  <li>Default cloud icon before a city is searched</li>
  <li>Fully responsive layout for mobile and desktop</li>
  <li>Fast, clean, and minimal interface</li>
</ul>
